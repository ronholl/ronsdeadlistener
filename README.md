# Ron’s Deadhead Listener

A simple web app to explore and listen to every Grateful Dead show (1965–1995).

## Features

- ⚡ Browse all known Grateful Dead concerts  
- 🎧 Stream shows via archive.org (Relisten integration)  
- 🌹 Track shows and mark your favorites  
- 🎲 Play a random show  
- 📅 “On This Day” show discovery  
- 🏆 Highlight standout performances  

## Dataset

- Curated master list of ~2,300 Grateful Dead shows  
- Deduplicated (one show per date)  
- Cleaned and structured for consistent playback  
- Designed to align with archive.org availability  

## Tech

- Single-page HTML app  
- No backend  
- Hosted via GitHub Pages  

## Data & Audio Sources

- Audio streams provided by archive.org via Relisten  
- This app does not host any audio files
